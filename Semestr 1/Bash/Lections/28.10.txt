Потоки:
0 - input
1 - output (терминал, консолька)
2 - errors

Например,
echo "hi" это 1
cat > file
...		это 0

cd ../
ls -l
можно заменить на
echo "hi" | cat > h			т.е. мы создаем файл h и записываем туда hi
echo 	- это 1
cat 	- это 0

cd /home/den && touch h 	-выведет ошибку
cd /home/den && touch h 2> /dev/null	-пропустит ошибку
/dev/null - пустая дириктория(черная дыра нафиг)
после 2> можно вводить любую дирикторию

Команда grep

regex - регулярные выражения
в grep:
.+ - 1 и больше
.? - 0 или 1
.* - 0 и больше

^ - начало строки
$ - конец строки
ls -l /home | grep -P ' m[a-z]+$' делаем поиск по логинам начинающимся на m
